Posiciones y despliegues de elementos

Las posiciones disponibles

  • static
  • relative
  • absolute
  • fixed
  • sticky

Static. Posición por default

Cajas con posición estática y display block

display inline-block

Posición relativa

Cajas con posición relativa, deja ocupada su antigua posición y se puede acomodar visualmente en cualquier parte.

Soy un span

Posición absoluta

Cajas con posición absoluta, ocupa la misma posición de su elemento padre con posición relativa. Si no encuentra un padre con posición relativa, se posiciona relativamente con < html >
También se puede acomodar en cualquier parte.

Posición fixed

Este artículo está ajustado al elemento < html > y no es afectado por el scroll.
También se puede acomodar en cualquier parte.

.class { position: fixed; }

Posición sticky

La posición sticky ajusta este elemento a un padre con posición relativa y dejará de ser afectado por el scroll cuando esté dentro de su elemento padre.
Las coordenadas se ajustan con respecto al elemento < html >

.class { position: sticky; }

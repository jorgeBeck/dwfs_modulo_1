Posición Relativa

Descripción de posición relativa: Ocupa el mismo espacio, y se puede acomodar visualmente en otro lado. Sus coordenadas se ajustan con respecto a su posición original.

Posición absoluta

Igual que posición relativa, solamente que pasa a ocupar el mismo espacio que su papá relativo, o < html > si no encuentra uno.
También se puede acomodar visualmente en donde queramos, a partir de su posición.

Posición sticky

La posición sticky ajusta este elemento a un padre con posición relativa y dejará de ser afectado por el scroll cuando esté dentro de su elemento padre.
Las coordenadas se ajustan con respecto al elemento < html >

.class { position: sticky; }
